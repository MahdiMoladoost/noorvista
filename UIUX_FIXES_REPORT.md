# NoorVista UI/UX Fixes — 3.0.1

اعمال‌شده روی نسخه ارسالی:

- نمایش مجدد تاپ‌بار تیره در موبایل، با ارتفاع ثابت ۴۴px و جلوگیری از پنهان شدن توسط لایه‌های قبلی موبایل.
- افزایش ارتفاع هدر روشن موبایل به ۷۲px، بهبود جای‌گذاری لوگو، ورود و دکمه منو.
- کاهش فاصله بالای هیرو/بج «کلینیک دکتر محمدصادق حق‌پرست» در موبایل.
- اصلاح کارت پزشک موبایل و مسیر تصویر دکتر با fallback، تا تصویر در کارت شیشه‌ای دیده شود.
- اصلاح چیدمان دسکتاپ تاپ‌بار: اطلاعات تماس/ساعت سمت راست، پیام‌رسان‌ها سمت چپ.
- پایدارسازی رندر تاپ‌بار تا با هر resize یا دریافت تنظیمات بی‌دلیل innerHTML بازنویسی نشود.
- پایدارسازی فوتر برای تلفن و ساعت؛ از بازنویسی HTML بدون تغییر جلوگیری شد.
- اصلاح حلقه/پرش تابع تبدیل اعداد فارسی: MutationObserver هنگام تبدیل اعداد موقتاً disconnect می‌شود تا تغییرات خودش را دوباره تحریک نکند.
- حذف load اجباری SDK نشان از HTML عمومی؛ SDK فقط وقتی API نقشه فعال و مختصات معتبر باشد به‌صورت lazy load می‌شود، و در خطا fallback نمایش داده می‌شود.
- fallback امن برای اطلاعات کلینیک در صورت 503/خطای تنظیمات عمومی: نام کلینیک، تلفن، آدرس و ساعت کاری خالی نمی‌مانند.
- تنظیم top منوی موبایل با درنظرگرفتن تاپ‌بار تیره + هدر روشن، برای جلوگیری از هم‌پوشانی.

## 3.0.2 topbar stability hotfix
- بازنویسی کامل topbar عمومی برای جلوگیری از flicker و ناپایداری آیکون‌ها.
- رفع تداخل اسکریپت تبدیل اعداد فارسی با ساختار topbar که باعث حذف موقتی آیکون تلفن می‌شد.
- چیدمان desktop: آدرس، ساعت و تلفن با آیکون در سمت راست؛ پیام‌رسان‌ها در سمت چپ.
- بهبود ارتفاع و تراز عمودی آیتم‌های topbar در desktop و mobile.


## 3.0.3 final topbar correction
- جایگذاری topbar با flex مستقل و direction:ltr برای جلوگیری از mirror شدن CSS Grid در RTL.
- Desktop: پیام‌رسان‌ها در چپ و آدرس/ساعت/تلفن در راست با آیکون و تراز عمودی ثابت.
- جلوگیری از دستکاری `.nv-topbar169` توسط اسکریپت عمومی تبدیل اعداد فارسی برای حذف کامل پرپر آیکون تلفن.
- نمایش پایدار topbar موبایل با ارتفاع 46px.


## 3.0.5 final cleanup
- جایگزینی کامل topbar با nv-final-topbar؛ ساختار قبلی، clone قبلی و source socials مخفی می‌شوند.
- حذف تکرار آیکون‌های پیام‌رسان با تولید فقط یک لینک برای هر پلتفرم و حذف URL تکراری.
- نمایش اجباری شماره تلفن با ارقام فارسی در topbar.
- محدود کردن اسکریپت عمومی متن/اعداد برای جلوگیری از پرپر footer و خراب‌شدن آیکون‌ها.
- اصلاح active state منوی اصلی؛ فقط لینک صفحه جاری انتخاب‌شده می‌ماند.
- کاهش فاصله خالی بالای hero صفحه اصلی.


## 3.0.6 footer stability fix
- فوتر بخش «اطلاعات کلینیک» از ساختار قدیمی contact-list جدا شد و به بلوک مستقل nv-footer306 تبدیل شد.
- contact-list و footer-socials قدیمی hidden شدند تا اسکریپت‌های runtime دیگر متن/آیکون‌های قابل مشاهده را بازنویسی نکنند.
- اسکریپت public-site-settings دیگر روی data-no-fa-digits و nv-footer306-info متن را جایگزین نمی‌کند.
- تلفن، ساعت، آدرس‌ها و شبکه‌های اجتماعی فوتر با markup پایدار و بدون Mutation loop رندر می‌شوند.


## 3.0.7 Persian digits everywhere
- اسکریپت تبدیل اعداد فارسی بازنویسی شد تا همه اعداد قابل‌مشاهده صفحه فارسی شوند.
- تبدیل اعداد بدون بازنویسی innerHTML و بدون حلقه پرپر انجام می‌شود.

## 3.0.8 static footer anti-flicker
- فوتر بخش اطلاعات کلینیک با همان ساختار نهایی contact-list پایدار شد.
