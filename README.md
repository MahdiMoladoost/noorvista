# NOORVISTA Clinic Management System

نسخه بازساختاردهی‌شده با ساختار استانداردتر برای Node.js + HTML/CSS/JS.

## نصب

```bash
npm install
cp .env.example .env   # در ویندوز: copy .env.example .env
npm run init-db
npm start
```

## مسیرهای مهم

- صفحه اصلی: `/`
- خدمات: `/services`
- نمونه صفحه خدمت: `/services/blepharoplasty.html`
- ورود: `/login`
- پنل مدیر: `/dashboard/admin`
- مسیر قدیمی سازگار: `/dashboard/panel/admin`
