# راه‌اندازی مرحله‌ای

## 1. نصب پیش‌نیازها

- Node.js 18 یا بالاتر
- MySQL 8 یا MariaDB سازگار

## 2. تنظیم دیتابیس

در فایل `.env` مقدارها را تنظیم کنید:

```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=رمز_مای‌اس‌کیوال
DB_NAME=noorvista_clinic
JWT_SECRET=یک_رشته_طولانی_امن
```

## 3. نصب پکیج‌ها

```bash
npm install
```

## 4. ساخت دیتابیس و داده تست

```bash
npm run setup
```

این دستور ابتدا جدول‌ها را با `init-db.js` می‌سازد و بعد با `seed-data.js` کاربران تست، کلینیک، پزشک، بیمار، خدمات، نوبت، صف، EHR و فاکتور نمونه را وارد می‌کند.

## 5. اجرای سرور

```bash
npm start
```

سپس:

```text
http://localhost:3000/login
```

## 6. تست سلامت API

```text
http://localhost:3000/api/v1/health
```

اگر MySQL وصل باشد خروجی `mysql: true` می‌دهد.

## امنیت تولید

- مقدار `JWT_SECRET` را حتماً تغییر دهید.
- HTTPS را فعال کنید.
- برای MySQL کاربر جدا با دسترسی محدود بسازید.
- پوشه `public/uploads` را پشت محدودیت MIME و آنتی‌ویروس قرار دهید.
- بک‌آپ منظم برای دیتابیس فعال کنید.


## افزودن خدمات آزمایشی چشم‌پزشکی بدون حذف داده‌ها

برای اضافه‌کردن یا به‌روزرسانی خدمات پایه چشم‌پزشکی روی پایگاه داده فعلی، از دستور زیر استفاده کنید:

```bash
npm run seed:services
```

این دستور هیچ جدول یا رکوردی را حذف نمی‌کند و خدمات را بر اساس شناسه متنی یکتا (slug) درج یا به‌روزرسانی می‌کند. برای این کار نیازی به اجرای `npm run init-db` نیست.
